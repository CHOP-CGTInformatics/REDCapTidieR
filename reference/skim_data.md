# Apply applicable skimmers to data

A helper function for
[`add_skimr_metadata()`](https://chop-cgtinformatics.github.io/REDCapTidieR/reference/add_skimr_metadata.md)
which applies applicable skimmers to a given dataframe.

## Usage

``` r
skim_data(redcap_data, redcap_metadata, is_labelled)
```

## Value

A dataframe
