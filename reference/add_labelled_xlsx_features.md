# Add labelled features to write_redcap_xlsx

Helper function to support `labelled` aesthetics to XLSX supertibble
output

## Usage

``` r
add_labelled_xlsx_features(
  supertbl,
  supertbl_meta,
  wb,
  sheet_vals,
  include_toc_sheet = TRUE,
  include_metadata_sheet = TRUE,
  supertbl_toc = NULL
)
```

## Arguments

- supertbl:

  a supertibble generated using
  [`read_redcap()`](https://chop-cgtinformatics.github.io/REDCapTidieR/reference/read_redcap.md)

- supertbl_meta:

  supertibble metadata generated by
  [`bind_supertbl_metadata()`](https://chop-cgtinformatics.github.io/REDCapTidieR/reference/bind_supertbl_metadata.md)

- wb:

  An `openxlsx2` workbook object

- sheet_vals:

  Helper argument passed from `write_redcap_xlsx` to determine and
  assign sheet values.

- include_toc_sheet:

  Include a sheet capturing the supertibble output. Default `TRUE`.

- include_metadata_sheet:

  Include a sheet capturing the combined output of the supertibble
  `redcap_metadata`. Default `TRUE`.

- supertbl_toc:

  The table of contents supertibble defined in the parent function.
  Default `NULL`.
