# Bind supertbl metadata

Simple helper function for binding supertbl metadata into one table.
This supports creating the metadata XLSX sheet as well as
`supertbl_recode`.

## Usage

``` r
bind_supertbl_metadata(supertbl)
```

## Arguments

- supertbl:

  A supertibble generated using
  [`read_redcap()`](https://chop-cgtinformatics.github.io/REDCapTidieR/reference/read_redcap.md)
