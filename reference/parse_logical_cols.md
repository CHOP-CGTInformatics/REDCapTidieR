# Convert yesno, truefalse, and checkbox fields to logical

Convert yesno, truefalse, and checkbox fields to logical

## Usage

``` r
parse_logical_cols(db_data, db_metadata, call = caller_env())
```

## Arguments

- db_data:

  A REDCap database object

- db_metadata:

  A REDCap metadata object

- call:

  call for conditions
