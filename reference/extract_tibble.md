# Extract a single data tibble from a REDCapTidieR supertibble

Take a supertibble generated with
[`read_redcap()`](https://chop-cgtinformatics.github.io/REDCapTidieR/reference/read_redcap.md)
and return one of its data tibbles.

## Usage

``` r
extract_tibble(supertbl, tbl)
```

## Arguments

- supertbl:

  A supertibble generated by
  [`read_redcap()`](https://chop-cgtinformatics.github.io/REDCapTidieR/reference/read_redcap.md).
  Required.

- tbl:

  The `redcap_form_name` of the data tibble to extract. Required.

## Value

A `tibble`.

## Details

This function makes it easy to extract a single instrument's data from a
REDCapTidieR supertibble.
